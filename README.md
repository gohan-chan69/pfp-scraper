# pfp-scraper
scrapes discord profile pictures from a guild fast

## Installation
- Click on the `pfp-scraper.zip`.
- Extract the zip.
- Open `cmd`.
- Open main.py with `py main.py` command 

## Usege
- Enter your user token in `line 27` of the code.
- enter channel id.
